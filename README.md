# 3DCoster

A free, open-source cost calculator for 3D printing sellers. Know your true cost per print and set profitable prices with confidence.

**[Try it now at 3dcoster.com](https://3dcoster.com)**

## Features

- **True Cost Calculation** - Filament, electricity, printer depreciation, nozzle wear, labor time - nothing is left out
- **Break-even Tracking** - Know exactly how many units you need to sell to recover your investment on each model
- **Marketplace Fees** - Etsy, Facebook Marketplace, Kijiji - see your actual profit after all platform fees
- **Asset Library** - Manage all your filaments, consumables, tools, and printers in one organized place
- **Shipping Calculator** - Local delivery gas costs, carrier rates, packaging materials - all factored in
- **Works Offline** - Desktop app works in your workshop with no internet. Your data stays on your machine

## Getting Started

### Use the Web App

Just visit [3dcoster.com/app](https://3dcoster.com/app) - no account required, completely free.

### Download Desktop App

Get the native desktop app for offline use at [3dcoster.com/download](https://3dcoster.com/download).

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Waarangel/3dcoster.git
cd 3dcoster

# Install dependencies
npm install

# Start development server
npm run dev
```

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Dexie (IndexedDB)
- Tauri (desktop app)
